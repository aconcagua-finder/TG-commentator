Что где настраивается

Раздел Для чего Когда идти
AI Провайдер по умолчанию и API ключи. Всегда сначала — без ключей генерация не пойдёт.
Аккаунты Добавление/проверка аккаунтов, сон, прокси и назначение роли. После AI — нужен хотя бы 1 авторизованный аккаунт.
Комментирование Список чатов, где бот будет комментировать; внутри — фильтры/триггеры/сценарии. Когда есть аккаунты — добавьте 1–2 чата и настройте.
Реакции Авто‑реакции (эмодзи) на новые посты. Опционально, если хотите «оживить» активность.
Мониторинг Отслеживание постов в каналах по промпту и уведомления вам. Опционально, если нужен «радар» по теме.
Прокси Хранилище прокси + проверка; назначаются на аккаунты. Опционально, если нужно распределять трафик или обходить лимиты.
Ручной запуск Добавляет задачу «обработать конкретный пост по ссылке» в очередь. Когда нужно отработать старый пост или проверить настройки.
Ребрендинг Меняет имена/аватары аккаунтов по теме или по донору. Опционально, обычно после того как всё стабильно работает.
Статистика Логи действий из SQLite и экспорт в Excel. После запуска — чтобы понимать, что и кто сделал.
