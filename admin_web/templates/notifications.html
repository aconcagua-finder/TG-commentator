Полный token из соображений безопасности сюда не подставляется обратно.
Кнопка «Показать» раскрывает только то, что ты вводишь сейчас в это поле.
Можно указать числовой chat_id, @username, публичную ссылку https://t.me/... или invite-ссылку закрытого чата.
Допустимые форматы
-1001234567890 @test23032206 https://t.me/test23032206 https://t.me/+AbCdEf123456
Для @username и публичной ссылки t.me/... авторизованные аккаунты не нужны: бот отправит сообщение напрямую. Для invite-ссылки закрытого чата нужен хотя бы один авторизованный аккаунт, потому что Bot API сам не умеет превращать invite в chat_id. Бот всё равно должен быть добавлен в группу или канал, иначе тестовая отправка не пройдёт.
Где взять данные
Token создаётся через @BotFather. Chat ID можно узнать, написав боту и вызвав Bot API getUpdates или через вспомогательного бота для просмотра ID чатов, но можно просто вставить ссылку на нужный чат.
